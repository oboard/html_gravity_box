# 物理引擎番茄可视化

一个基于 Matter.js
物理引擎的互动时间数据可视化项目。通过物理引擎模拟的气泡展示不同事件的持续时间，支持触摸和重力感应交互。

## 功能特点

- 🎨 自适应大小
- 📱 支持触摸和鼠标交互
- 🔄 响应设备方向变化
- 🌓 支持深色模式
- 📊 自动计算并展示时间统计
- 💫 平滑的物理动画效果

## 技术栈

- Matter.js - 2D 物理引擎
- JavaScript ES6+
- HTML5 Canvas

## 安装和使用

1. 安装依赖：

```bash
bun install
```

2. 启动开发环境：

```bash
bun run dev
```

3. 访问 http://localhost:5173 查看效果。
